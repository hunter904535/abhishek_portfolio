## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/hunter904535/abhishek_portfolio.git
```

### Navigate to Project

```bash
cd abhishek_portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
